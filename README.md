# 📊 Customer Segmentation and Purchase Analysis

## 📌 Overview
This project focuses on cleaning, processing, and analyzing customer purchase data for a retail business. The goal is to transform raw customer records into structured information that can support segmentation, spending analysis, and category-based customer insights.

## 🎯 Objective
To prepare customer data for analysis and identify meaningful customer segments based on age, spending behavior, and purchase categories.

## 🛠️ Tools & Technologies
- Python
- Data Cleaning
- Lists and Functions
- Conditional Logic
- Basic Customer Segmentation

## 📂 Dataset
The dataset contains customer-level purchase information stored in nested lists. Each record includes:
- User ID
- Customer name
- Age
- Favorite purchase categories
- Spending amounts by category

## 🔍 Key Steps
- Cleaned customer names by removing extra spaces and formatting inconsistencies
- Split full names into structured name components
- Converted ages into integer format
- Standardized category names to lowercase
- Built a reusable function to clean user records
- Processed the full customer dataset into a structured format
- Calculated total company revenue
- Identified customers under 30 years old
- Filtered younger high-spending customers
- Detected customers who purchased in specific categories
- Created a reusable function to retrieve clients by category

## 📈 Key Results
- Total revenue calculated from the cleaned dataset: **9,189**
- Identified a young customer segment for targeted campaigns
- Detected high-value young customers for potential loyalty actions
- Enabled category-based filtering for customer behavior analysis

## 💡 Business Impact
This type of analysis can help a retail company:
- Segment customers by age and spending behavior
- Identify valuable customers for targeted promotions
- Analyze product category preferences
- Support personalized marketing strategies
- Build a cleaner foundation for future reporting and analytics
